## Control statements

#### if, elif, else

```
n = int(input())
if n == 1:
    print("Equal")
elif n > 1:
    print("Greater")
else:
    print("Smaller")
```

```
s = "Hello"
c = "H"
if c in s:
    print("Yes")
else:
    print("No")
>>> Yes
```

#### for

n = 10
for n in range(10):
    print(n)
